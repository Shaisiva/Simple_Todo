# Simple Daily Todo Chrome Extension

This is a standalone Chrome new-tab todo extension inspired by the Simple Todo layout.

## Features

- Replaces the Chrome new tab page.
- Three task sections: Today, This week, Eventually.
- Add, edit, delete, and complete todos.
- Remove completed todos.
- Email summary icon that opens an email draft to `sivaelango.periyasam@granicus.com`.
- Settings modal with light, dark, and system theme.
- Accent color selection.
- Optional uploaded background image.
- Data is saved locally in the browser.

## Load in Chrome

1. Open Chrome.
2. Go to `chrome://extensions/`.
3. Enable `Developer mode`.
4. Click `Load unpacked`.
5. Select this folder: `Todo-plugin`.
6. Open a new tab to see the todo app.

## Upload Preparation

Before publishing to Chrome Web Store:

1. Test all features using `Load unpacked`.
2. Add production extension icons if required by the Chrome Web Store.
3. Zip the contents of this `Todo-plugin` folder.
4. Upload the zip in the Chrome Web Store Developer Dashboard.
